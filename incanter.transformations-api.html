flatten-with

  function

  
Usage: (flatten-with f nested-map)


  
takes an arbitrarily deeply nested map, and flattens it to one level by merging keys. 

 (flatten-with str {:a {:b {:c 1}}}) -> {":a:b:c" 1}

  function

  
Usage: (levels-deep m)
